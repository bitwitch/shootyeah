# Shootyeah

A simple shmup game.  

[Play wasm version here](https://bitwitch.github.io/shootyeah/)  

### Dependencies
[SDL2](https://www.libsdl.org/download-2.0.php)

### Linux
To build: `make`  

### WASM version
Requires the [Emscripten compiler](https://emscripten.org/docs/getting_started/downloads.html)  
To build: `make wasm`
